Research Interests

My research focuses on:

  • Memory formation, retrieval, and distortion mechanisms
  • Decision making processes and their neural basis
  • Emotional influences on cognition and behavior
  • Aphantasia and visual imagery differences in memory

Current Projects

  • High-density fMRI multimodal memory dataset development
  • Investigating image-word association processes using cued recall tasks
  • Studying memory reconstruction and reactivation in free recall
  • Comparing brain patterns in individuals with aphantasia vs. typical visual recall
  • Examining factors influencing visual false memory and forgetting
